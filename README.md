# Start Using WebDriver Examples

This is the mini-sized package to start use WebDriver with Firefox and Chrome

## Pre-requisite:
1. MacOS or Linux or Windows
2. Java 8
3. Maven
4. Firefox Installed
5. Chrome Installed
